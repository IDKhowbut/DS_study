- 요청방식
(1) get
    1) 하이퍼링크 요청방식, url요청방식 (주소직접입력, 뒤에 ?age=30이란 조건도 넣어서 할 수 있음, but 데이터가 다 노출된다!!)
    2) 간단한 데이터만 전송 : body가 아닌 header에 데이터를 담고 있어 공간적인 한계가 있다.  (body에 담는 건 post방식)
    3) 데이터 노출 (보안에 취약) : 민감하지 않은 데이터 사용에 사용되어야 함
    4) 조회를 요청
    5) 기본 요청 방식
(2) post
    1) 폼 전송 방식
    2) 많은 양의 데이터 전송 가능
    3) 데이터 전송 시 노출되지 않음
    4) 데이터 변경이나 등록 요청시
