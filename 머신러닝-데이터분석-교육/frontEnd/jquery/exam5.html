jQuery란?

jQuery는 가볍고, 빠르며, 간결한 오픈소스 라이브러리 이다. 이를 이용하면 Rich Web UI를 개발하는 데 도움이 되는 다양한 기능들 즉, HTML문서 트래버스, 이벤트 처리, 애니메이션, Ajax 상호작용 등을 지원하며 빠르고 견고하게 리치 웹 어플리케이션을 개발할 수 있도록 지원한다.

이러한 jQuery의 기능적인 특징은 다음과 같다.

  • 막강한 CSS Selector
  • 크로스 브라우저 지원
  • 메서드 체인
  • Ajax 지원
  • 풍부한 플러그인 지원
